# URL Shortener API

Сервис для сокращения ссылок с возможностью аналитики и управления.

## 🚀 Функционал

### Обязательные функции (7.5 баллов)
1. ✅ Создание / удаление / изменение / получение информации по короткой ссылке
2. ✅ Статистика по ссылке (клики, дата создания, последнее использование)
3. ✅ Кастомные ссылки (уникальный alias)
4. ✅ Поиск по оригинальному URL
5. ✅ Время жизни ссылки (expires_at)

### Дополнительные функции (1.5 балла)
1. ✅ Удаление неиспользуемых ссылок (через N дней)
2. ✅ История истекших ссылок

### Кэширование (1 балл)
- ✅ Кэширование популярных ссылок в Redis
- ✅ Кэширование статистики
- ✅ Очистка кэша при обновлении/удалении
